# Auctor Web App Prototype

Investor-ready static web app prototype for **Auctor**, an AI ecosystem that maps physical problem-solving behavior into hiring, staffing, training, and workforce intelligence.

## What is included

- Employer dashboard for hiring and staffing recommendations
- The Lab assessment dashboard for physical challenge packs
- Employee Cognitive DNA profile view
- Auctor internal operations dashboard
- AI assistant examples for configuration and explainability
- Self-serve configuration flow
- Interactive pricing and consulting calculator
- Productized consulting model
- Validation, fairness, and benchmark sections
- Responsive design for desktop and iPhone-sized screens

## Run locally

```bash
npm start
```

Then open:

```text
http://127.0.0.1:4173
```

If PowerShell blocks `npm`, run:

```bash
node server.js
```

## Deploy on GitHub Pages

This is a static site. You can deploy it directly from the repository root.

1. Push these files to GitHub.
2. Go to repository `Settings`.
3. Open `Pages`.
4. Set source to `Deploy from a branch`.
5. Select the branch and `/root`.
6. Save.

GitHub Pages will serve `index.html` automatically.
